# Technology Stack

## Backend
- **Language:** C#
- **Runtime:** .NET 10
- **Framework:** ASP.NET Core (Web API) for the API layer and .NET Worker for background processing.
- **Background Jobs:** Hangfire for managing and executing asynchronous tasks.

## Data Persistence
- **Database:** SQL Server
- **ORM:** Entity Framework Core (EF Core) for database interactions and migrations.

## Tooling & Infrastructure
- **Containerization:** Docker (via docker-compose.yml) for local development and deployment.
- **API Documentation:** Swashbuckle (Swagger) for interactive API documentation.
